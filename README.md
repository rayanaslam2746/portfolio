Personal Portfolio

An interactive portfolio website built around a 3D Rubik's Cube navigation system.

Overview
The site uses a 3D cube as the primary navigation element. As users scroll, the cube rotates through different faces, each representing a section of the portfolio. When scrolling stops, the cube settles on the nearest face and displays the corresponding content.
